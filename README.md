## How to Run (in order вЂ” each part depends on the previous one's output)

```bash
pip install -r requirements.txt --break-system-packages

python3 prepare_data.py                                   # -> data/raw_merged.csv
cd part1 && python3 part1_cleaning_eda.py && cd ..         # -> data/cleaned_data.csv
cd part2 && python3 part2_regression_classification.py && cd ..
cd part3 && python3 part3_ensembles_pipeline.py && cd ..   # -> best_model.pkl
cd part4 && cp .env.example .env                           # then paste your API key into .env
python3 part4_llm_explanation.py && cd ..                  # requires best_model.pkl
```

## Dependency / Version Notes

All packages are pinned in `requirements.txt` to the exact versions this project was built and
verified against (Python 3.12.3, scikit-learn 1.8.0, pandas 3.0.2, numpy 2.4.4). If you use a
different pandas major version, note that `pd.read_json(..., lines=True)` and `df.astype("category")`
behavior is stable across 1.x/2.x/3.x for the operations used here, but exact `float`/`Int64`
formatting in printed logs may differ slightly вЂ” this does not affect any numeric result, only
cosmetic output formatting.
